# Privacy Policy

## Overview
This application, *message-tracker*, is a private internal tool
used to measure and analyze response times for Instagram Business Direct Messages.

## Data Access
The application accesses Instagram messaging data only from Instagram Business
accounts that are explicitly authorized and added as testers to the application.

The data accessed may include:
- Message timestamps
- Sender and receiver identifiers
- Message metadata required for response-time calculation

## Data Usage
The collected data is used solely for:
- Measuring response times
- Internal analytics and evaluation

## Data Storage and Sharing
- No personal data is sold or shared with third parties
- Data is not used for advertising or marketing
- Data is stored only for internal testing and analysis purposes

## Scope
This application is currently in a development and testing phase and is not
available for public use.
