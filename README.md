# message-tracker

A simple tool to track time taken for replying to DM on Instagram.

## How it works:

-User/Customer sends a message to the Instagram Business account
-Timestamp is recorded and stored in memory
-Business replies (identified using is_echo: true)
-Response time is calculated
-Data is stored in MongoDB:

This repository contains the Privacy Policy and Terms of Service
for the Instagram Response Time Tracker application.

The application is a private internal tool used for testing and analytics.
